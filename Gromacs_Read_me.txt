Steps to execute the script
----------------------------

1. Create a sh file in your terminal using "Gromacs_install.sh" and copy the script to that file and save it.
    or 
   Copy the Gromacs_install.sh file given to your terminal.

2. Give the user/group/others permission to execute the sh file by using the command "chmod +x Gromacs_install.sh"

3. Run the script file now by using the below command.
 
      "./Gromacs_install.sh "
